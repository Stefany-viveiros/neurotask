## 🚀 Sobre o projeto

O **NeuroTask** é uma aplicação fullstack que utiliza Inteligência Artificial para tornar o gerenciamento de tarefas mais inteligente, automatizado e eficiente.

Diferente de uma lista comum, o sistema:
- Analisa suas tarefas  
- Define prioridades automaticamente  
- Gera insights inteligentes  
- Oferece um assistente virtual integrado  

---

## ✨ Funcionalidades

- ✅ Criar, editar e excluir tarefas  
- 🤖 Classificação automática de prioridade com IA  
- 💡 Geração de insights inteligentes  
- 💬 Chat com assistente de IA  
- ⚡ Sistema de XP (gamificação)  
- 📊 Dashboard de produtividade  
- 💾 Armazenamento local (LocalStorage)  
- 🎨 Interface moderna (Glassmorphism + Dashboard)  

---

## 🧠 IA integrada

A aplicação utiliza a API da OpenAI para:

- Classificar tarefas automaticamente (high, medium, low)  
- Gerar sugestões personalizadas  
- Fornecer respostas no chat inteligente

## ⚙️ Como executar

### 1. Clone o projeto


git clone https://github.com/Stefany-viveiros/neurotask.git

cd neurotask


### 2. Instale as dependências


npm install


### 3. Configure a API

Crie um arquivo `.env`:


OPENAI_API_KEY=sua_chave_aqui


### 4. Inicie o servidor


npm start


### 5. Abra no navegador

Abra o arquivo `index.html`
